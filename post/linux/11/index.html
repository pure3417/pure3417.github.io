    cat 文件  # concatenate的缩写，功能描述：查看文件内容，从第一行开始

  • 参数说明

  • 经验技巧

    一般查看比较小的文件，一屏幕能显示全的。

more：分屏查看内容

more 指令是一个基于VI 编辑器的文本过滤器，它以全屏幕的方式按页显示文本文件的内容。more 指令中内置了若干快捷键，详见操作说明。

  • 基本语法

    1

    more 文件名  # 功能描述：分屏查看文件内容

  • 操作说明

less：分屏显示文件内容

less 指令用来分屏查看文件内容，它的功能与more 指令类似，但是比more 指令更加强大，支持各种显示终端。less 指令在显示文件内容时，并不是一次将整个文件加载之后才显示，而是根据显示需要加载内容，对于显示大型文件具有较高的效率。

  • 基本语法

    1

    less # 功能描述：分屏查看文件内容

  • 操作说明

> 输出重定向内容 » 追加内容

>与 >>常搭配ls，cat，echo使用

  • 常用搭配

    1
2
3
4

    ls -l > 文件  # 功能描述：列表的内容覆盖到文件中
ls -al >> 文件  # 功能描述：列表的内容追加到文件末尾
cat 文件1 > 文件2  # 功能描述：将文件呢1得内容覆盖到文件2
echo "内容" >> 文件  # 功能描述：将输出内容追加到文件中
